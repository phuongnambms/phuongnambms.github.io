BUILDING CONTROL UNIT (BCU)

  • October 19, 2022
  • Phuong Nam

Introduction

The Central Controller (Building Control Unit) is a specialized central control unit in the BMS system integrated in buildings. The device is pre-installed with Control BMS Software that allows users to control their BMS system from any device that supports Web browser such as computer, phone, tablet ....

The device integrates communication standards via TCP IP and RS485 networks, allowing connection to other devices to perform signal reception, control, logging, extracting to excel files, drawing graphs, etc.

Feature

  • Support 1 communication port 485.
  • Supports 1 Ethernet port to read and write data from other devices.

Specification

Power Supply 24VAC ± 10% or 24VDC ± 5%.. Supply current >=3A
LAN communication Yes
Protcol Modbus RTU 485, Modbus IP, BACnet IP
Control interface Yes
Operating temperature limit From 0°C to 50°C
Operating humidity limit Less than 90% and no dew
Environment Operate in a cool, dry place without corrosive, flammable and explosive agents
Delivery and storage Store in a dry place, away from moisture. Storage temperature from 25°C to 70°C, humidity 5% to 95% and no dew
Dimension 119 x 100 x 56 ( mm )

Pins diagram

  • 24V IN and 0V IN: Power supply for BCU 24VAC/DC
  • A+ and B-: RS485 signal
  • LAN: Ethernet connection
